# Pipeline ETL - Santander Dev Week (Simulado)

Este projeto demonstra a construção de um pipeline ETL (Extract, Transform, Load)
utilizando Python e arquivos CSV, simulando o fluxo de dados da Santander Dev Week.

## Etapas do ETL

### Extração
Os dados são lidos a partir de um arquivo CSV com informações fictícias de usuários.

### Transformação
Para cada usuário, é gerada uma mensagem personalizada.

### Carregamento
As mensagens são salvas em um novo arquivo CSV.

## Como executar

```bash
python etl.py
